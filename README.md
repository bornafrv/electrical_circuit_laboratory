# 🔬 Electrical Circuit Laboratory – University of Tehran

This repository contains a comprehensive collection of experiments conducted in the **Electrical Circuit Laboratory** course at the **University of Tehran**.  
Each folder includes simulations, measurements, and detailed analyses for fundamental electrical concepts using real and simulated data.

---

## 📁 Directory Structure & Experiments

| Folder Name                                     | Description                                                                 |
|--------------------------------------------------|-----------------------------------------------------------------------------|
| ⚡ Transient Responses of First-Order RC         | Analyze charging/discharging curves, time constant (τ), and square wave behavior |
| ⚡ Transient Responses of Second-Order RC        | Study underdamped, overdamped, and critically damped responses using RLC |
| 📡 Frequency Responses of First-Order RC Circuit | Examine amplitude & phase response, cutoff frequency, and filtering behavior |
| 🔀 Non-Linear Circuit                            | Investigate diode V-I characteristics, Zener behavior, and load-line analysis |
| 🔧 Resistance Circuit                            | Validate Thevenin's theorem, power transfer, and superposition in resistive networks |
| 🎛️ Operational Amplifier                        | Implement inverting, summing, and buffer op-amp circuits with gain/response analysis |
| 📉 Oscilloscope Introduction                     | Learn waveform stability, trigger modes, and signal synchronization |
| 🧮 Digital Multimeter Introduction               | Measure internal resistance, simulate voltmeter behavior, and assess error |
